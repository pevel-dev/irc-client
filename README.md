## Irc client
#### Авторы
* Шеметов Павел
* Иудинов Михаил

### Фичи
* Работает по протоколу irc
* Асинхронно
* Gui pyqt6
* Получение списка каналов
* Подключение к каналу
* Отправка любых по длине сообщений

### Usage
```shell
pip install -r requirements.txt
python src/irc.py
```

### Test
Установите зависимости из ```dev-requirements.txt```
Запустите
```shell
pytest
```
